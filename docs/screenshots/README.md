# Screenshots Directory / 截图目录

This directory contains application screenshots for documentation.  
此目录用于存放应用程序界面截图，供文档使用。

## Naming Convention / 命名规范

| Filename             | Description                 | 说明     |
| -------------------- | --------------------------- | ------ |
| `main-window.png`    | Main application window     | 主应用窗口  |
| `trading-module.png` | Trading info module         | 交易信息模块 |
| `statistics.png`     | Statistics analysis view    | 统计分析视图 |
| `pie-chart.png`      | Portfolio pie chart         | 投资组合饼图 |
| `settings.png`       | Preferences/settings window | 设置窗口   |
| `dark-theme.png`     | Dark theme view             | 深色主题视图 |
| `cash-module.png`    | Cash management module      | 现金管理模块 |
| `select-coin.png`    | Coin selection dialog       | 币种选择界面 |

## How to Add Screenshots / 如何添加截图

### Option 1: Direct Upload / 方式1：直接上传

1. Take screenshots of the application / 截取应用界面截图
2. Place them in this directory / 将截图放入此目录
3. Update README.md to reference them / 更新 README.md 引用这些截图

### Option 2: Using Git / 方式2：使用 Git

```bash
# Add screenshot files / 添加截图文件
git add docs/screenshots/*.png

# Commit / 提交
git commit -m "docs: Add application screenshots"

# Push to GitHub / 推送到 GitHub
git push origin master
```

## Recommended Image Specifications / 推荐图片规格

- **Format**: PNG (recommended) or JPG / 格式：PNG（推荐）或 JPG
- **Resolution**: 1920x1080 or 1280x720 / 分辨率：1920x1080 或 1280x720
- **File Size**: Try to keep under 500KB per image / 文件大小：尽量控制在 500KB 以内
- **Content**: Show typical use cases and key features / 内容：展示典型使用场景和核心功能
